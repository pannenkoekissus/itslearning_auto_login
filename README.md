# Itslearning auto login
This repo let's you set credentials to fill in automatically and choose to press the sign in button automatically on the Itslearning login website
# Installation instructions
1. Clone this repository or download it as zip (click on code, then download zip). Extract the zip file if needed
2. Go to chrome://extensions (This obviously works for chrome, but this usually works for any browser, sometimes browsers might have their own website for this, usually "your browsers name"://extensions
3. Toggle on developer mode in the top right
4. Click load unpacked in the top left
5. Browse to the location where you cloned this repository or you unpacked the zip
6. Click open
7. Pin the extension by pressing the extension (puzzle piece) button to the right of the adress bar and clicking the pin icon next to the extension
8. Set your credentials via the popup
9. Enjoy!
